## Filosofía Elegida: Dark Luxury / Neo-Gothic Premium

**Design Movement**: Dark Luxury / Neo-Gothic Premium

**Core Principles**:
1. Elegancia a través de la restricción — cada elemento gana su lugar
2. Dorado como señal de calidad, no de exceso
3. Tipografía serif para autoridad, sans-serif para claridad
4. El espacio negativo es parte del diseño

**Color Philosophy**: Fondo casi negro (#0b0b0b) que evoca un estudio de tatuajes de alta gama. El dorado (#bfa15a) se usa con moderación para acentuar, no saturar. Texto en blanco cálido (#f0ede8) para suavidad, grises neutros para jerarquía.

**Layout Paradigm**: Columnas asimétricas en desktop, flujo vertical nativo en móvil. Secciones alternadas entre #0b0b0b y #0f1113 para crear ritmo visual sin bordes duros.

**Signature Elements**:
1. Líneas doradas delgadas como divisores y acentos
2. Glow sutil dorado en el logo (drop-shadow)
3. Bordes con opacidad reducida (rgba gold) en tarjetas

**Interaction Philosophy**: Hover states sutiles (scale 1.05, gold border), transiciones de 200-300ms. Botones con feedback táctil claro (active:translateY). Lightbox con blur backdrop.

**Animation**: Framer Motion con ease personalizado [0.16, 1, 0.3, 1] para entradas suaves. Stagger de 80-120ms entre elementos de lista. Scroll-triggered con viewport margin.

**Typography System**: Playfair Display (400, 600, 700) para títulos — evoca lujo editorial. Inter (300, 400, 500, 600) para cuerpo — legibilidad perfecta en pantalla.
